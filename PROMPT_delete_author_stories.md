# Prompt: удалить историю из конструктора

Добавить в авторское рабочее место Story Builder полноценное удаление истории:

- В backend сделать `DELETE /api/author/stories/{storyId}`.
- Перед удалением проверить, что текущий `X-Player-Id` является владельцем истории.
- Удалять связанные прохождения, сцены, выборы, ассеты и файлы загруженных ассетов.
- В списке историй конструктора добавить аккуратную кнопку удаления рядом с редактированием и статистикой.
- Перед удалением показывать понятное подтверждение с названием истории.
- После успешного удаления обновлять список историй и сбрасывать сохраненный `lastImportedStoryId`, если удалена текущая открытая история.
- Покрыть сценарии тестами: владелец может удалить историю, чужой автор получает `403`.
